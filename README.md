# Ansa — website

The marketing site and embedded AI-receptionist widget for **Ansa**, the AI
receptionist for local service businesses.

- **Live:** https://thatqne.github.io/ansa/
- The chat bubble in the corner is the live demo (runs in local mode by default).
- To connect the real engine, set `data-api="https://your-engine"` on the widget
  `<script>` in `index.html`.

Auto-deploys to GitHub Pages on every push (see `.github/workflows/pages.yml`).
